## 🛠 技术方案总结

经过多轮测试与逆向分析，我们最终采用了 **“CDP 凭证同步 + 自主 POST 请求”** 的混合方案：

1. **解决鉴权难题**：通过 Chrome DevTools Protocol (CDP) 实时提取浏览器中复杂的 `Cookie` 和 `X-XSRF-TOKEN`。
2. **解决数据缺失**：确认为 `POST` 接口，通过模拟真实的 JSON Body（包含 `record_type_v4` 等字段）获取全量数据。
3. **解决翻页限制**：API 单次上限 30 条。脚本实现了 `page_index` 循环逻辑，可一次性抓取数百条历史记录。
4. **低干扰运行**：无需刷新页面，不影响浏览器正常使用，无确认弹窗。
