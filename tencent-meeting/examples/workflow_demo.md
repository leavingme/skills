# 腾讯会议自动化典型流程示例

以下演示如何串联使用该 Skill 获取一份完整的会议纪要。

## 场景：自动下载最近一场会议的转写

1. **列出最近会议**
   运行 `list_recordings.py` 找到第一条记录。
   ```text
   1. [2026-03-16 14:30] 产品需求讨论会
      录制ID: 2033460555300589569 | 会议ID: 12400565953094634593
   ```

2. **监控转写进度**
   运行 `task_monitor.py`，等待输出：
   ```text
   ✅ 转写状态: 已完成 (Done) (Code: 0)
   ```

3. **执行导出任务**
   将 ID 填入 `export_file.py` 并运行：
   ```bash
   python scripts/export_file.py
   ```
   获得文件：`minutes_12400565953094634593.txt`

4. **后续处理**
   将得到的 `.txt` 文件交给 AI 助手进行摘要提取或动作行动项生成。
