# AI Agent Skills 集合

本项目是一个专为 AI 智能助手开发的 **Skills (技能)** 集合仓库。目前核心收录了 **本牛云盘编辑器** 相关能力。

## 核心技能：本牛云盘专辑管理器 (`benew-album-manager`)

该技能提供了对本牛云盘（Benewtech）音频资产的全面自动化管理能力。

### 核心功能 ✨

- **文件夹与专辑管理 (CRUD)**：
  - 支持列出根目录文件夹、新建、重命名或删除文件夹。
  - 支持在指定文件夹下新建专辑、更新专辑信息及删除专辑。
- **音频自动同步 (Sync)**：
  - 监听本地文件夹，自动识别未上传的 MP3 音频。
  - 自动获取音频时长并在云端注册。
  - 内置文件名标准化功能（自动清洗重复标记如 `(1)`）。
- **自动化深度优化 (Optimize)**：
  - **智能去重**：按序号或名称保留最早版本，清理重复副本。
  - **极致排序**：识别序号（如 `001`）自动修正乱序。
  - **统一配置**：批量为专辑音轨统一更新封面键值，保持视觉一致。
- **安全凭证抓取**：
  - 内置 CDP 协议监控，能通过控制当前 Chrome 实例静默抓取最新 Cookie 和 Family ID，实现零配置上手。

## 用法指南 🚀

### 1. 作为 AI Agent 技能使用 (推荐)

如果你正在使用具备 Antigravity 执行环境的 AI 助理，你可以直接通过自然语言与其交互：

> - "帮我看看云盘根目录下有哪些文件夹？"
> - "在 '测试目录' 下新建一个叫 '2026合集' 的专辑，并将本地下载目录的音频同步过去。"
> - "清理一下专辑内的重复项并修正排序。"

AI 将自动识别 `SKILL.md` 中的指令，通过 `scripts/` 下的工具链为你自动化完成任务。

### 2. 交互式配置引导 (Cold Start)

当你第一次在没有 `.env.benew` 配置文件的环境下运行时，AI 会自动通过以下流程引导你：
1. **凭证抓取**：尝试自动接管你的 Chrome 以获取登录态。
2. **目标发现**：列出云端文件夹供你选择，而不是强迫你寻找 ID 链接。
3. **持久化保存**：配置完成后会自动生成本地临时配置文件，以便下次直接执行。

## 安全与隐私 ⚠️

- **凭证安全**：所有抓取到的 Cookie 和配置参数均保存在本地 `.env.benew` 中，并已被 `.gitignore` 排除，**严禁提交至公共仓库**。
- **资产保护**：重要文件夹（如核心资源目录）在代码层面做了过滤逻辑，防止误删除。
