# 本牛云盘专辑管理器 (Benew Album Manager)

本牛云盘专辑管理器是一个用于自动化管理本牛云盘（Benewtech）音频专辑的工具集合。它可以帮助用户将本地的 MP3 文件自动同步到云盘中，并提供专辑数据优化功能（如去除重复音轨、根据编号重置排序、统一更新音轨封面等）。

## 核心功能 ✨

- **自动同步 (Sync)**：扫描本地监控文件夹，对比云端已存在的列表。把尚未存在于云端的临时本地方 MP3 音频自动上传并注册为专辑音轨，省去了大量手动录入繁琐流程。内置通过 macOS 命令获取真实播放时长。
- **自动优化 (Optimize)**：
  - **去重**：自动检测序号或名称相同的音轨，保留最早上传的版本，剔除多余由于中断等原因创建的临时副本。
  - **智能排序**：自动识别前缀中用于区分集数的 3 位数字序号（如 `001`, `002`），以修正因跨越时间段陆续上传所引起的乱序问题。
  - **统一配置**：批量去掉无意义重命名尾缀格式，自动为整个专辑内的所有音轨强制更换指定的封面。
- **单文件上传演示 (Upload API)**：提供了一个基础范例，演示本牛云盘如何结合七牛云进行分步骤文件处理和登记。

## 用法指南 🚀

### 方式一：使用内置的 AI Agent 助理技能 (推荐)

如果你正在全自动使用具备 Antigravity Context 环境的系统，此项目本身内置成了 Agent Skill：
可以直接在聊天界面向 AI 求助要求执行相应任务，并提供一次对应的运行环境参数：

> "帮我把 `~/Downloads/.../` 目录下的音频同步到本牛云盘，并做一次优化"

AI 助理将自动读取/询问所需的鉴权及坐标变量信息（包括 `COOKIE`, `FAMILY_ID`, `ALBUM_ID` 等），使用 `scripts/` 下的版本为你自动化搞定一切任务，避免把隐私写入硬盘导致泄露。

### 方式二：本地单纯运行 Python 脚本

你可以把项目 clone 到本地直接当做普通 Python 任务脚本进行触发。这要求在对应的脚本的 `================ 配置区 ================` 修改常量信息为你的真实参数：

1. **设定常量环境**
   - `COOKIE`: 你登录本牛云盘网页版后通过 F12 Header 获取的 Cookie (`connect.sid=...`)
   - `FAMILY_ID`: 你账号所在的固定家庭标识
   - `ALBUM_ID`: 对应要监控/管理的目标专辑识别码
   - `WATCH_DIR`: 你用来向专辑投放更新等音频的本地绝对物理路径检测口
   - `ALBUM_COVER`: 指定的在线封面直链（优化排序统一时专用）

2. **开始同步新增本地项：**
   ```bash
   python benew_sync.py
   ```
   （获取云端名单过滤 $\rightarrow$ 七牛云 Uptoken 获取 $\rightarrow$ 逐个直传 $\rightarrow$ 本牛回算时长并注册生效）

3. **优化与整理乱序图册内容：**
   ```bash
   python album_optimizer.py
   ```
   （按集数组合寻找副本 $\rightarrow$ 物理删除多出多余的部分 $\rightarrow$ 以序号重定向位置排序 $\rightarrow$ 更换全部封面归档保存）

## 注意事项 ⚠️

- 本项目基于非公开请求协议逻辑实现，针对网页端接口特性进行了拟合调用。如未来官方改版认证形态，不排除会发生变动。
- 本地调试抓包获得有效的 `COOKIE` 凭据，由于携带账号直通权限，请务必只在安全的私有沙盒环境存储执行，避免泄露！
